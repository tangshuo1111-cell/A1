## 当前结构

当前评测体系已经落成以下分层：

- `V0`：Schema Smoke + Sandbox
- `V1`：Route + Exit State + Basic Honesty
- `V2`：Capability Chains + Fake Success Negatives
- `V2.5`：Multi-turn State Closure
- `V3`：Complex / Agent Collaboration
- `V4`：Report + Regression Gate

## 当前治理口径

- 评测层是外挂治理层，不是第二套业务系统。
- 收的是脆弱性，不是判定强度。
- 硬风险继续强判，脆弱观测默认软判。
- `known_issue` 只记录真实产品缺陷。
- `backend_unavailable / case_timeout / execution_error` 不进入 known issue。
