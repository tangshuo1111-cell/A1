## 当前治理口径

- 评测体系完成，不等于产品能力完全正确。
- `known issue failure` 不等于 V4 失败。
- `backend_unavailable / case_timeout / execution_error` 不等于产品缺陷。
- 收的是脆弱性，不是判定强度。
- 硬风险继续强判，脆弱观测默认软判。
