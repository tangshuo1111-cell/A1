# 用户可见状态文案

目的：统一 **主对话区摘要** 与 **ContextRail 详版** 的中文表达，与 `turn_exit_gate` 出口字段对齐。

治理与实现（单一事实源）：

| 层级 | 落点 |
| --- | --- |
| 产品口径 | 本文（`docs/pm/10_用户可见状态文案.md`） |
| 前端 SSOT | `frontend/lib/contextMeta/statusCopy.ts`（`task_status` / `pending_kind` / lane 中文） |
| 主区摘要 | `TurnStatusSummary` → `buildContextLines` → `statusCopy` |
| 异步/任务区 | `ChatAsyncStatus` → `statusCopy`（禁止组件内再写一套映射） |
| 单元测试 | `frontend/lib/contextMeta/statusCopy.test.ts` |
| 后端字段 | `backend/application/chat/turn_exit_gate.py`（只产枚举，不写用户文案） |

**规则**：改用户可见中文时，先改 pm/10 与 `statusCopy.ts`，再跑 `npm run test`；不要在 `extractors.ts` 或 chat 组件里散落硬编码。

---

## 主对话区（2–3 行摘要）

| 行 | 来源字段 | 用户文案示例 |
| --- | --- | --- |
| 路径 | `extra.lane` / `extra.mode` | 「知识库直答 · 快速处理」 |
| 状态 | `task_status` | 「已完成」/「部分完成」/「处理中」 |
| 说明 | `pending_kind` / `insufficient_evidence` | 「已升级到复杂分析」/「证据偏薄，回答偏保守」 |

---

## task_status

| 内部值 | 用户文案 |
| --- | --- |
| `done` / `succeeded` | 已完成 |
| `partial` | 部分完成 |
| `failed` | 未完成 |
| `queued` | 排队中 |
| `running` | 处理中 |
| `pending` | 等待中 |

---

## lane（主路径）

| 内部值 | 用户文案 |
| --- | --- |
| `kb` | 知识库直答 |
| `document` | 文档理解 |
| `web` | 网页读取 |
| `video` | 视频处理 |
| `general` | 通用回答 |

---

## pending_kind / 升级原因

| 内部值 | 用户文案 |
| --- | --- |
| `escalate_to_complex` | 已切换到深度分析 |
| `escalate_to_async` | 已转入后台任务 |
| `more_kb_material` | 需要补充知识库材料 |
| `more_web_material` | 需要补充网页材料 |
| `more_document_material` | 需要补充文档材料 |
| `more_video_material` | 需要补充视频材料 |

---

## 失败 / 降级语义（soft vs hard）

| 内部语义 | 用户文案 | 类型 |
| --- | --- | --- |
| budget 熔断 / deadline | 处理时间较长，已返回当前可用结果 | soft |
| quality 升级 complex | 正在深度分析… | soft |
| approval 阻塞 | 请确认是否保存到知识库 | soft |
| 硬失败 | 本次未能完成，请稍后重试 | hard |
| `insufficient_evidence=true` | 证据偏薄，回答偏保守 | soft |

---

## 语义检索 / health.embedding.mode

| mode | 用户文案 |
| --- | --- |
| `index_present` | 语义索引已就绪 |
| `enabled_no_rows` | 语义索引为空，当前可能使用关键词检索 |
| `disabled` | 关键词检索（语义检索已关闭） |
| 其他 | 索引状态检查中 |

---

## quality_gate_passed（产品指标，默认不在主区展示）

| 值 | 含义（对内/侧栏） |
| --- | --- |
| `true` | 质量门通过或未触发阻断 |
| `false` | 质量门未通过（可能已升级或 partial） |
