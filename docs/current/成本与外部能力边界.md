# 成本与外部能力边界

本文档对应 `backend/config/cost_rule.py` 中的 `COST` 单例。
所有限制均可通过同名环境变量覆盖。

## 规则总表

| 类别 | 字段 | 默认值 | 环境变量 | 超限行为 |
|------|------|--------|----------|----------|
| LLM | ai_max_calls_per_turn | 5 | AI_MAX_CALLS_PER_TURN | 停止继续调用，返回降级回答 |
| LLM | ai_max_input_chars | 16000 | AI_MAX_INPUT_CHARS | 截断输入或拒绝 |
| LLM | max_output_chars | 4000 | MAX_OUTPUT_CHARS | 截断输出 |
| LLM | max_estimated_cost_usd | 0.05 | MAX_ESTIMATED_COST_USD | 拒绝并提示"需确认" |
| RAG | knowledge_max_chunks | 5 | KNOWLEDGE_MAX_CHUNKS | 只取前 N 条 |
| RAG | rag_max_top_k | 12 | RAG_MAX_TOP_K | 高质量模式上限，超过需显式配置 |
| 搜索 | web_search_max_results | 5 | WEB_SEARCH_MAX_RESULTS | 只取前 N 条结果 |
| 搜索 | web_search_max_query_chars | 200 | WEB_SEARCH_MAX_QUERY_CHARS | 截断查询 |
| 网页 | web_fetch_timeout_sec | 15 | WEB_FETCH_TIMEOUT_SEC | 超时返回失败 |
| 网页 | web_fetch_max_pages | 5 | WEB_FETCH_MAX_PAGES | 拒绝抓取更多页面 |
| 网页 | web_page_max_chars | 30000 | WEB_PAGE_MAX_CHARS | 截断正文 |
| 工具 | tool_max_steps | 8 | TOOL_MAX_STEPS | 停止工具链执行 |
| 视频 | video_fetch_timeout_sec | 90 | VIDEO_FETCH_TIMEOUT_SEC | 超时返回失败 |
| 视频 | video_asr_auto_seconds | 900 (15min) | VIDEO_ASR_AUTO_SECONDS | 15分钟内自动处理 |
| 视频 | video_asr_confirm_seconds | 3600 (60min) | VIDEO_ASR_CONFIRM_SECONDS | 15-60分钟需确认 |
| 视频 | video_asr_max_seconds | 3600 (60min) | VIDEO_ASR_MAX_SECONDS | 超过硬上限直接拒绝 |
| 上传 | upload_max_mb | 50 | UPLOAD_MAX_MB | 拒绝上传 |
| OCR | ocr_max_pages | 20 | OCR_MAX_PAGES | 只处理前 N 页 |
| 任务 | task_timeout_sec | 120 | TASK_TIMEOUT_SEC | 超时终止任务 |
| 任务 | session_max_tasks | 1 | SESSION_MAX_TASKS | 拒绝新任务 |

## 设计原则

1. **任何昂贵行为都不能静默发生** — 超限必须显式报错
2. **环境变量覆盖** — 生产/开发/测试可使用不同阈值
3. **分层防护** — LLM 次数 + 费用估算 + 工具步数三层保底
4. **视频三档** — auto(15min) / confirm(15-60min) / reject(60min+)
